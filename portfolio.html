Over the summer of 2018 at COSMOS, I created this 3D sound simulating app that helps military recruits train in a virtual sound environment, helping them get accustomed to the loud artillery noises, so that once in an actual battle they can focus on their mission and target. Using MetaWear C sensors, the program adjusts the positions of the various sounds based on the user's head position. Here is the link to the github repository

I created this app in XCode with Swift without having ever taken a course or being taught Swift, learning how to use the software from online tutorials. I implemented two different playing modes: multiplayer and single player. In the multiplayer mode, the user can grab a friend to play with them on the same screen. For single player, the user chooses a CPU level ranging from an easy opponent who chooses randomly to an unbeatable opponent.
